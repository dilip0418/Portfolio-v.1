A Web Bases Application for Detecting Plagiarism in the Document

"This was my final year project which is a dynamic web application that excels at detecting plagiarism in text data. Leveraging cutting-edge technology and AI, the application was built using Python's FastAPI framework, implementing NLP and VSM concepts. By integrating external APIs like scraper and Google's custom search, we efficiently find and analyze the best matched results online. The application generates a report of plagiarism at the end which contains link to matching content present online(Internet) with the respective matching ratio. The sleek and intuitive frontend, crafted with HTML, CSS, and a touch of JavaScript, ensures a seamless user experience throughout."

A GUI-Based Desktop Application for Movie Data Management System

This project presents a user-friendly GUI-based Desktop Application designed for efficiently managing movie-related data. Developed using Python's Tkinter library and powered by a MySql database, the application enables seamless storage and organization of movie information. Notably, it incorporates a module that fetches comprehensive details about actors, cast members, production houses, and other related data from Wikipedia.

A Web Based Application for logging status of rented Instruments in Musical Instruments Shop

The Musical Instruments Rental Status Accounting System is a web application developed using Java Servlets, JDBC, and MySQL. It simplifies the instrument rental process and stock management for shopkeepers by automating rental status updates, return date tracking, and instrument stock management. The user-friendly interface, designed with HTML, CSS, and Bootstrap, ensures a smooth and efficient experience. It also gives the collection statistics for daily and monthly records, helps shopkeepers to improve their financial performance.
